URL Shortening Microservice


Add a new URL

thiswebpage/new/your url

Ex. https://thisapp.herokuapp.com/new/www.google.com


List saved urls in json

thiswebpage/showurls

Ex. https://thisapp.herokuapp.com/showurls


Go to shortened url page

thiswebpage/go/shortenedurlnumber

Ex. https://thisapp.herokuapp.com/go/1234
